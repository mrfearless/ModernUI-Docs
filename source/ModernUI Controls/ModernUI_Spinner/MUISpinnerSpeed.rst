.. _MUISpinnerSpeed:

========================
MUISpinnerSpeed 
========================

MUISpinnerSpeed, hWin::ref:`MUIWND<MUIWND>`, MillisecSpeed:MUIVALUE

Sets the spinner animation speed (in milliseconds)

**Parameters**

* [in] **hWin** - handle to the ModernUI_Spinner control
* [in] **MillisecSpeed**

**Return**

None

**Example**

::

   Invoke MUISpinnerSpeed, hSpinner, 100
